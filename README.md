## 🛠️ Fitur Utama & Nilai Plus Aplikasi
1. **Autentikasi Pengguna (Nilai Plus):** Fitur Register akun baru & Login dinamis yang terhubung langsung dengan `tbl_user` di MySQL.
2. **Manajemen Data (CRUD):** Fitur Tambah, Ubah, Hapus Data Baju di Toko Pakaian.
3. **Pencarian Otomatis:** Filter pencarian data secara real-time saat mengetik di kolom pencarian tanpa tombol tambahan.
4. **Cetak Laporan (Nilai Plus):** Ekspor otomatis seluruh daftar tabel produk pakaian ke format dokumen PDF/Printer.
5. **Arsitektur Clean Code:** Menggunakan design pattern Model-View-Controller (MVC) dan Data Access Object (DAO).

## 🗄️ Cara Menjalankan Aplikasi
1. Import file database `produk.sql` yang tersedia di dalam repositori ini ke phpMyAdmin Anda.
2. Buka project menggunakan NetBeans IDE.
3. Jalankan Clean and Build project terlebih dahulu.
4. Jalankan aplikasi utama melalui file `LoginForm.java`.
